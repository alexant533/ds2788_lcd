程序用 avrstudio6 编写 
内部晶振为8M

主程序  ds2788_lcd.c 调用lcd显示在lcd2002显示上，调用2788的读写程序，读取ds2788读写电池的特性， 每隔一段时间读取并更新显示。显示部分分为三个页面，第一个页面为温度，电流，电压显示页面，不可以编辑，第二个页面里可以设置外接的负载电阻阻值，两个按键同时按下可以进入，退出编辑模式，在编辑模式里左键可以移动光标，右键可以改变数值，第三个页面里可以设置满电池容量和当前电池容量， 两个按键同时按下可以进入，退出编辑模式，在编辑模式里左键可以移动光标，右键可以改变数值。
 
显示2002程序  lcd.c和lcd.h lcd.h里定义了lcd的控制管脚控制命令和数据管脚数据命令以及lcd的读写函数  lcd.c里是具体lcd控制的实现函数 


读写ds2788程序 ds2788.c和ds2788.h  ds2788.h里定义了单总线的管脚操作，包括输入输出，拉高拉低，以及 ds2788的寄存器register地址和操作码registercode
，ds2788的读写函数定义。ds2788.c实现了读写操作和复位操作。

读取按键程序　key.c和key.h key.h里定义了读取两个按键的操作，当按下第一个按键时，get_key()返回1,当按下第二个按键时,get_key()返回2,当两个同时按下时返回3，当没有按键按下时，返回0。key.c里实现了读取按键的操作。
